# 🛒 Grocery E-Commerce Web App

A full-stack modern eCommerce web application built using the MERN stack.  
This project allows users to browse products, add them to cart, place orders, and manage their account, while admins can manage products, orders, and user interactions.

---

## 🚀 Features

### 👤 User Features
- User Registration & Login (JWT आधारित authentication)
- Browse products with details
- Add to Cart & Wishlist
- Checkout with Cash on Delivery (COD)
- Order placement & tracking
- View order history
- Invoice generation
- Contact form submission

---

### 🛍️ Product Features
- Product listing with images
- Product details page
- Category-based filtering
- Search functionality
- “You May Also Like” recommendation system:
  - Based on category
  - Cart items
  - Wishlist
  - Previous orders

---

### 🤖 Smart Features
- Recommendation engine (multi-stage logic)
- Floating chatbot UI:
  - Pre-built queries
  - Product suggestions
  - Fallback support message

---

### 🛠️ Admin Features
- Admin login
- Add / Edit / Delete products
- View all orders
- View user messages
- Manage product inventory

---

### 🎨 UI/UX Features
- Modern responsive design (Tailwind CSS)
- Interactive Hero Banner
- Smooth animations (Framer Motion)
- Global scroll-to-top button
- Mobile-friendly layout

---

## 🧑‍💻 Tech Stack

### Frontend
- React (Vite)
- Tailwind CSS
- Context API (Cart & Wishlist)
- Framer Motion

### Backend
- Node.js
- Express.js
- MongoDB (Mongoose)

### Other Tools
- Cloudinary (Image Upload)
- Multer (File handling)
- JWT (Authentication)
